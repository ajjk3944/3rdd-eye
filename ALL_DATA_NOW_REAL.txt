📊 কিভাবে REAL DATA পাচ্ছে:
-----------------------------

1. WiFi Networks:
   ```typescript
   const scanResult = await wifiScanner.scan()
   // Returns: all available networks with SSID, BSSID, RSSI, frequency
   ```

2. Device Info:
   ```typescript
   const deviceData = await deviceInfoPlugin.getDashboard()
   // Returns: model, manufacturer, deviceType
   ```

3. SIM Info:
   ```typescript
   const simData = await deviceInfoPlugin.getSimInfo()
   // Returns: SIM slots, carrier names, signal strength, network type
   ```

4. Ping & Packet Loss:
   ```typescript
   // Ping gateway
   const pingResult = await pingGateway(gatewayIP)
   
   // Calculate packet loss from 5 pings
   const packetLoss = await calculatePacketLoss(gatewayIP)
   ```

5. Network Config:
   ```typescript
   const netResult = await netScanner.getNetworkInfo()
   // Returns: IP, gateway, DNS, subnet mask
   ```
